# 1. 🛒 LUXE - 프리미엄 쇼핑몰

모던한 디자인의 이커머스 웹 애플리케이션

**📍 위치:** `shop-app/`

#### 주요 기능
- 🏪 **상품 관리**
  - 12개 프리미엄 패션 아이템
  - 카테고리별 필터링 (상의/하의/아우터/신발/액세서리)
  - 다양한 정렬 옵션 (가격순/평점순/리뷰순)
  - 실시간 검색 기능
  
- 🛍️ **쇼핑 경험**
  - 상품 상세 모달 (색상 선택, 수량 조절)
  - 위시리스트 (하트 버튼)
  - 호버 시 장바구니 담기 버튼 애니메이션
  - 할인율 및 뱃지 표시 (BEST/NEW/SALE)
  
- 🛒 **장바구니**
  - 우측 슬라이드 드로어 UI
  - 수량 변경 / 개별 삭제 / 전체 비우기
  - 실시간 합계 계산
  - 3만원 이상 무료 배송 안내
  
- 🎨 **디자인**
  - 미니멀하고 세련된 UI
  - 부드러운 애니메이션 효과
  - 반응형 레이아웃 (모바일 2열 그리드)
  - Blur 효과의 sticky 헤더

#### 기술 스택
- React 18 (함수형 컴포넌트 + Hooks)
- Context API (장바구니 상태 관리)
- CSS Modules
- Vite
- Unsplash 이미지

#### 실행 방법
```bash
cd shop-app
npm install
npm run dev
```

#### 프로젝트 구조
```
shop-app/
├── src/
│   ├── components/      # UI 컴포넌트
│   │   ├── Header       # 헤더 + 검색 + 장바구니 버튼
│   │   ├── Hero         # 메인 배너
│   │   ├── ProductCard  # 상품 카드
│   │   ├── ProductGrid  # 상품 목록 + 필터
│   │   ├── ProductModal # 상품 상세 모달
│   │   ├── CartDrawer   # 장바구니 드로어
│   │   └── Footer       # 푸터
│   ├── context/         # 전역 상태 관리
│   │   └── CartContext  # 장바구니 Context
│   ├── data/            # 상품 데이터
│   │   └── products.js  # 12개 상품 정보
│   └── App.jsx          # 메인 앱
```

## 🚀 공통 실행 환경

### 요구사항
- Node.js 16.x 이상
- npm 또는 yarn

### 개발 서버 실행
각 프로젝트 폴더에서:
```bash
npm install    # 의존성 설치
npm run dev    # 개발 서버 실행
npm run build  # 프로덕션 빌드
```
